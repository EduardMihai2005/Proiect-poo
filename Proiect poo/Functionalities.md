# Functionalitati Proiect - Gestiune Abonamente

1. **Gestionarea Revistelor**
   - Adaugare revista noua (Fizica sau Digitala) - date citite din fisier sau tastatura.
   - Vizualizare lista reviste folosind operatori supraincarcati.
   - Calcul automat al numarului total de reviste (static member).

2. **Gestionarea Utilizatorilor**
   - Incarcare utilizatori din fisier text.
   - Adaugare manuala utilizator.

3. **Gestionarea Abonamentelor**
   - Creare abonament (asociere User - Revista).
   - Validare existenta ID-uri folosind functii Template.
   - Monitorizare status (activ/expirat).
   - Simulare trecere timp (scadere luni ramase).

4. **Concepte POO Implementate**
   - Mostenire & Polimorfism (Clasa abstracta Revista).
   - Incapsulare (Date private/protected).
   - STL (Vectori).
   - Template-uri (Functie cautare generica).
