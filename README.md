# Proiect POO - Sistem Abonamente Reviste

## Descriere
Aplicatie C++ pentru gestionarea abonamentelor la reviste fizice si digitale.

## Functionalitati Cheie
- **Incarcare Automata**: La pornire, datele sunt citite din `reviste.txt` si `utilizatori.txt`.
- **Polimorfism**: Revistele sunt tratate generic, dar afiseaza detalii specifice (Transport vs Link).
- **Template**: Cautarea in liste se face printr-un mecanism generic.

## Cum se ruleaza
1. Se deschide proiectul in Visual Studio.
2. Se asigura prezenta fisierelor .txt in folderul sursa.
3. Se compileaza si ruleaza `main.cpp`.
